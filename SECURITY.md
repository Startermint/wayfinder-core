# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

Report security issues through GitHub Security Advisories for this repository. Include:

- Affected version or commit
- Steps to reproduce
- Expected impact
- Any known workaround

We will acknowledge valid reports, investigate, and coordinate a fix before public disclosure.

## Supported Versions

Security fixes target the latest released version of Wayfinder Core unless otherwise noted in the release notes.
